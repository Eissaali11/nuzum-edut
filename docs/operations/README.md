# Operations & How-tos

This folder contains operational guides for running benchmarks and smoke tests.

Quick pointers:
- Benchmarks: `scripts/run_dashboard_benchmark.py`
- Smoke tests: `scripts/test_attendance_smoke.py` and `scripts/check_attendance_endpoints.py`
- Unit tests (attendance logic): `scripts/run_attendance_unit_tests.py` or `pytest tests/test_attendance_logic.py`
