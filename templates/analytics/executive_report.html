Power BI Dashboard Layout Guide | دليل تخطيط Power BI

Step 1: Import Data

Load the exported Excel file (8 sheets) into Power BI Desktop using "Get Data" → "Excel Workbook"

Step 2: Create Relationships

In Model View, establish relationships:
• FACT_Financials[employee_key] → DIM_Employees[employee_key]
• FACT_Maintenance[vehicle_key] → DIM_Vehicles[vehicle_key]
• FACT_Attendance[employee_key] → DIM_Employees[employee_key]

Step 3: Dashboard Layout

Top Row (KPI Cards): Total Salary Liability, Active Fleet %, Workforce Count, Attendance Rate
Middle Row: Financial Heatmap (Matrix), Regional Map (Filled Map), Department Treemap
Bottom Row: Maintenance Trend (Line Chart), Salary by Region (Bar Chart), Fleet Health (Donut)

Step 4: Advanced Features

• Add slicers for: Region, Department, Date Range
• Create DAX measures for YoY growth, running totals, and forecasts
• Enable drill-through pages for employee and vehicle details
• Configure bookmarks for different executive views
