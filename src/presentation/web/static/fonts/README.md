# خطوط العرض الموديولي (نُظم)

هذا المجلد يخدم قالب **presentation/web** (الهيكل الموديولي الجديد).  
ملف **theme.css** يحمّل الخط من المسار النسبي: `../fonts/beIN-Normal.ttf` أي هذا المجلد.

## الملف المطلوب

| الملف | الحالة |
|-------|--------|
| **beIN-Normal.ttf** | يجب وضعه هنا |

## ماذا أفعل؟

انسخ ملف الخط **beIN-Normal.ttf** من مجلد الخطوط الرئيسي للمشروع إلى هنا:

- **من:** `d:\nuzm\static\fonts\beIN-Normal.ttf`
- **إلى:** `d:\nuzm\presentation\web\static\fonts\beIN-Normal.ttf`

أو انسخه من أي مصدر لديك إلى هذا المجلد بالاسم بالضبط: **beIN-Normal.ttf**.

بدون هذا الملف، المتصفح سيستخدم الخط الاحتياطي (Cairo) في صفحات العرض الجديد.
