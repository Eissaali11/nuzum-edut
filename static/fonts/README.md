# خطوط المشروع

## الخط الافتراضي: beIN-Normal

المشروع مضبوط لاستخدام خط **beIN-Normal** كخط افتراضي في كل الواجهات.

**لماذا لا يظهر الخط؟**  
لأن ملف الخط غير موجود في هذا المجلد. المتصفح يحاول تحميل الملف من المسار التالي ولا يجده فيستخدم الخط الاحتياطي (Cairo).

### ما الذي تحتاج فعله؟

1. احصل على ملف الخط **beIN-Normal.ttf** (من مصدرك الرسمي أو الترخيص الذي لديك).
2. ضع الملف في هذا المجلد بالاسم التالي بالضبط:
   ```
   beIN-Normal.ttf
   ```
   المسار الكامل:
   ```
   d:\nuzm\static\fonts\beIN-Normal.ttf
   ```
3. حدّث الصفحة (أو أعد تشغيل التطبيق إن لزم). بعدها سيُحمّل الخط تلقائياً وسيظهر في كامل المشروع.

### ملفات الخطوط الحالية في المجلد

- `arialbd.ttf` — موجود
- `beIN-Normal.ttf` — **مفقود** (يجب إضافته لتفعيل الخط الافتراضي)
